Test script for bidict.collapsingbidict::

    >>> from bidict import collapsingbidict

Initializing with collapsing mappings succeeds::

    >>> collapsingbidict([(1, 1), (2, 2), (1, 2)])
    collapsingbidict({1: 2})

Collapsing updates succeed::

    >>> b = collapsingbidict({1: 1, 2: 2})
    >>> b[1] = 2
    >>> b
    collapsingbidict({1: 2})
    >>> b = collapsingbidict({1: 1, 2: 2})
    >>> b[:2] = 1
    >>> b
    collapsingbidict({1: 2})
    >>> b = collapsingbidict({1: 1, 2: 2})
    >>> b.update({1: 2})
    >>> b
    collapsingbidict({1: 2})
